# Praveen P - Personal Portfolio

A modern, responsive, and highly interactive personal portfolio website built to showcase my skills, projects, internships, and achievements. Designed with a focus on premium aesthetics and user experience.

![Portfolio Preview](./public/vite.svg)

**Live Demo:** [Praveen's Portfolio](https://praveen-p-portfolio.vercel.app/) *(Update this link to your actual deployment URL)*

## ✨ Features

- **Dark/Light Mode Toggle:** A fully integrated dark mode that saves user preferences locally.
- **Infinite Scrolling Projects:** A custom animated carousel that smoothly cycles through full-stack project cards.
- **Modern UI & Aesthetics:** Built using Tailwind CSS, featuring glassmorphism elements, vibrant gradient backgrounds, dynamic hover animations, and the sleek 'Outfit' Google Font.
- **Working Contact Form:** The "Leave a Message" form is directly integrated with [FormSubmit](https://formsubmit.co/), ensuring all messages are delivered straight to my email inbox without the need for a backend server.
- **Interactive Animations:** Powered by `framer-motion` for smooth scrolling reveals and staggered component loads.

## 🛠️ Tech Stack

- **Frontend Framework:** React.js (v19)
- **Styling:** Tailwind CSS (v3)
- **Build Tool:** Vite
- **Animations:** Framer Motion
- **Icons:** React Icons

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js installed on your machine.
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/PraveenP9105/praveen-portfolio.git
   ```

2. **Navigate into the project directory**
   ```sh
   cd praveen-portfolio
   ```

3. **Install dependencies**
   ```sh
   npm install
   ```

4. **Start the development server**
   ```sh
   npm run dev
   ```

5. **Open in Browser**
   Navigate to `http://localhost:5173` in your web browser to view the application.

## 📬 Contact Form Setup

The contact form is configured to use [FormSubmit](https://formsubmit.co/). 
To activate it for your email:
1. Ensure the email address in `src/components/ContactForm.jsx` is correct.
2. Submit a test message through the form locally or in production.
3. Check your email inbox and click the **Activate** button in the email sent by FormSubmit. 
4. The form is now fully functional and will route all future messages to your inbox.
